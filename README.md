# Travel Diary · 旅行档案

一个记录旅途、城市与回忆的个人旅行日记。

## 启动

```bash
npm install
npm start
```

默认启动后打开 [http://localhost:9000](http://localhost:9000)。

如要指定端口：

```bash
node js/server.js --port 8080
```

如果启动时报 `pyftsubset` 不存在，先安装字体工具：

```bash
pip install "fonttools[woff]"
```
